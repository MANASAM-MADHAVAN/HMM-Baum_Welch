## Project Description

This project implements the Hidden Markov Model (HMM) using the Baum-Welch algorithm.

The model learns:

- Transition Matrix (A)
- Emission Matrix (B)
- Initial Probability Distribution (π)

The algorithm iteratively updates model parameters to maximize the likelihood of the observation sequence.
